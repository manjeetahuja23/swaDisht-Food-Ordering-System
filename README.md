# swaDisht - Restaurant Management System

## Overview

swaDisht is a full-stack restaurant management and food ordering platform developed as a capstone project.

The system allows customers to browse restaurants, order food online, manage orders, and track purchases while providing administrators with powerful tools to manage restaurants, menus, users, and orders.

The project demonstrates full-stack web development using PHP, MySQL, HTML, CSS, JavaScript, and Bootstrap.

## Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/swaDisht-Restaurant-Management-System.git
```

### Setup Database

1. Create a MySQL database.
2. Import SQL files from the SQL folder.
3. Update database credentials in the connection files.

### Run Project

1. Place project inside XAMPP htdocs folder.
2. Start Apache and MySQL.
3. Open:

```text
http://localhost/swaDisht
```
